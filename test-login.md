## 테스트: 로그인 상태 UI 변경

### 1️⃣ 회원가입/로그인 흐름
1. http://localhost:8092/signup.html 방문
2. 회원가입 → localStorage에 userId 저장
3. 자동으로 closet.html로 이동
4. 네비게이션 바에서:
   - "로그인" 버튼 사라짐 ❌
   - "사용자이름" 표시 ✅
   - "로그아웃" 버튼 생성 ✅

### 2️⃣ 로그아웃
- "로그아웃" 버튼 클릭
- localStorage 초기화
- index.html로 이동
- 네비게이션 바에 "로그인" 버튼 다시 나타남

### 3️⃣ 옷장 기능
- 로그인 상태: 옷 추가/수정/삭제 모두 가능 ✅
- 비로그인 상태: login.html로 리다이렉트 ✅

### 예상 결과
- ✅ 회원가입 → 자동 로그인 상태
- ✅ 모든 페이지에서 로그인 상태 표시
- ✅ 로그아웃 시 로그인 페이지로 이동
- ✅ 비로그인 상태에서 옷장 페이지 접속 → 로그인 페이지로 리다이렉트
